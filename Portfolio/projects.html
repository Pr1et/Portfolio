Airbnb Data Analysis

This project analyzes Airbnb listing data to uncover insights about pricing trends, popular locations, availability, and customer preferences. The analysis helps identify factors affecting listing prices and occupancy rates using exploratory data analysis and visualization techniques.

Technologies Used:

  • Python
  • Pandas
  • Numpy
  • Matplotlib
  • Seaborn
  • Jupyter Notebook

Key Features:

  • Performed data cleaning and preprocessing
  • Analyzed pricing trends across different locations
  • Visualized availability and booking patterns
  • Identified factors influencing Airbnb listing prices
